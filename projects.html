Enterprise Limit Management System [ELMS]

Finflux | Jan 2023 - July 2023

  • An application which calculates the exposure of individuals and groups to flag the lending system if exposure constraints are met. Solely responsible for the implementation of all calculation formulae.
  • Provides both API call for single transaction and SFTP File Handling for bulk transactions using a cron job.
  • Third-party integration for security (including token generation) and user management.

Image Captioning using Neural Networks

National Institute of Engineering | Jan 2021

  • Modified and trained a pre-trained Natural Language Processing model InceptionResNetV2(RNN) on the Flickr8k dataset to generate captions. Calculated accuracy by computing BLEU scores.
  • Converted generated captions to speech using Google Text-to-Speech API. Hosted on a website using AWS EC2 instance.
